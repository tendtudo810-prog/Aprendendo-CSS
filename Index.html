Aprendendo CSS


O que é o CSS?

O CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é a linguagem usada para definir a aparência de páginas web, separando o conteúdo (HTML) da apresentação visual. Com ele, é possível controlar cores, fontes, espaçamentos, posicionamento e muito mais, tornando o design consistente e fácil de manter.


Tags importantes do CSS:

  • Seletor : indica qual elemento HTML será estilizado.
  • Propriedade : define o estilo específico que será aplicado ao elemento.
  • Valor : determina o valor da propriedade, como uma cor ou tamanho de fonte.
